Educational App - Case Study

Learn First, Pay Later
An EdTech UX Case Study

This project focuses on redesigning an EdTech course platform to address a key user problem users often pay for courses without knowing whether the content is truly worth the money. The solution introduces a 50% free access model, allowing learners to experience a substantial portion of a course before making a purchase decision, which helps build trust and confidence. The redesign uses a modern dark-themed interface with clear visual hierarchy and transparent access indicators to clearly distinguish free and locked content, ensuring users can easily understand what they can access. Overall, the project aims to improve user trust, enhance the learning experience, and support informed decision-making before course enrollment.

Train Booking App Redesign

Reducing Cognitive Load in Train Booking Apps
through Clear Route Guidance

This project focuses on redesigning a train booking app to reduce cognitive load and improve route clarity for users. The existing experience presented too much information on a single screen and failed to clearly differentiate between direct and indirect routes, especially when no direct train was available. The redesign prioritizes essential information, introduces a clear visual hierarchy, and presents indirect journeys as a single, continuous flow with explicit train-change points. The goal was to guide users step by step, make seat availability easier to understand, and help users confidently choose the right route with minimal effort.
